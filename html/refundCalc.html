Refundable Bottles

When drink containers are purchased, consumers pay a deposit fee on them. This fee can be refunded by returning the item(s) to a certified recycling depot.

As of Oct 1, 2020, deposits on drink containers over 1L were changed from 20¢ to 10¢. Items with a deposit are:

Glass:
  • Glass bottle, any size
  • Glass alcohol container, any size
Metal:
  • Aluminum can, any size
  • Aluminum alcohol can, any size
  • Bi-metal can, any size
Drink Box:
  • Drink Box, any size
  • Gable Top, any size
  • Tetra Pak box, any size
  • Bag-in-a-box, any size
  • Bag-in-a-box alcohol container, any size
Plastic:
  • Plastic bottle, any size
  • Plastic Liquor container, any size
  • Drink pouch, any size
Note:

Milk and milk substitute drink containers do not have a deposit fee on them when purchased at a store, and therefore do not have a refund; they are outside the legislated deposit/refund system. However, they can be recycled normally at various depots or by curb-side pickup.
